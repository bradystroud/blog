# Blog

Built with Next.js, TinaCMS and Tailwind.

## Features

- **Responsive Design**: Tailwind CSS, mobile-first
- **Content Management**: TinaCMS (MDX blogs, JSON-driven pages)
- **SEO**: Sitemap generation, RSS feed, dynamic OG images via `@vercel/og`

## Dev

1. Clone
2. `bun install`
3. Copy `.env.example` to `.env.local` and fill in TinaCMS variables
4. `bun dev`
